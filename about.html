About Bookoria

Bookoria is a simple digital library system designed to make reading easier and more accessible for everyone. Whether you're looking for popular international bestsellers or inspiring Indonesian literature, Bookoria lets you browse, search, and manage books efficiently.

Currently, I'm 17 years old and this is my first ever working website. This project was created as a beginner-friendly web development exercise to practice HTML, CSS, JavaScript, and JSON data handling. It also supports user login and book management features.

I aim to grow this platform into a more complete digital library system prototype that supports user borrowing, personalized recommendations, and more.
